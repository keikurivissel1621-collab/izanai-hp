# 実演動画（画面録画）の置き場

cc-demo.html はこのフォルダの mp4 を参照します。
以下のファイル名で mp4 を置くと自動で再生されます（無い間はプレースホルダ表示）：

- ad-optimization.mp4   … 広告運用・最適化
- mail-reply.mp4        … メール対応・自動化
- accounting.mp4        … 経理処理・自動仕訳
- spreadsheet.mp4       … スプレッドシート分析
- doc-create.mp4        … 資料作成・提案書

ポスター画像（任意）：同名の .jpg を置くと再生前のサムネになります（例: ad-optimization.jpg）
