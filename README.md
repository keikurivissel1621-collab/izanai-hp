# MUKIAI 公式サイト（株式会社むきあい）

AI伴走支援・Claude Code 導入基盤設計支援の公式ホームページ（静的サイト）。

## 構成
| ファイル | 内容 |
|---|---|
| `index_rebuild.html` | MUKIAI 本体トップ（AI伴走支援） |
| `claude-code.html` | Claude Code 導入基盤設計支援（連続LP） |
| `cc-about / cc-diff / cc-demo / cc-cases / cc-pricing .html` | Claude Code サブページ |
| `lp.html` | 3分AIセキュリティ診断 |
| `assets/` | CSS・JS・ロゴ・実写画像・実演動画 |

## デプロイ
GitHub に push すると Netlify が自動デプロイします（ビルド不要・`netlify.toml` 参照）。

## 実演動画
`assets/videos/` に MP4 を配置すると `cc-demo.html` で再生されます（ファイル名は `assets/videos/README.md` 参照）。
